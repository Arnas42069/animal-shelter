WINDOWS CMD terminale paleisti komandas:

```bash
ssh-keygen -t ed25519
type %USERPROFILE%\.ssh\id_ed25519.pub
```

Visus laukus palikti tuščius
Nukopijuotą pilną raktą atsiųsti
